CIS 598: Computer Science Senior Project

A project based off MCON

Welcome to my page about CIS 598 Spring 2012. Please look below to find out more information about my project.

I want to thank Eugene Vasserman for taking the time to be my project adviser this semester.

Abstract

Membership Concealment Overlay Network (or MCON) is an anonymous and censorship-resistant system consisting of a large network of nodes that are connected by physical links. My part in this project is to simulate a situation where a new node is invited by multiple members of MCON. This new node needs a map of how the network is laid out. Creating a map of the network is a difficult process because the network conceals the exact topology information, however each node conceals its identity by encrypting its identification information before sending it on, which reveals no real information about the exact identity of any members. Through interprocess communication, the source floods its physical neighbors, and sends a discovery message through a series of intermediate nodes until it reaches its destination creating a logical link from the source node to the destination node. This is done by simulating flooding and networking through the Python programming language. Each node is its own process, and during the flood, it communicates through Queues to pass messages between one another. Before processing a message and continuing with the flood, each node must keep track of the previous hop and the next hop to tell each node whether or not it has seen the flood message before. This allows for all possible routes to be constructed for the network topology map.
